This repo contains all the settings that I want to be able to synchronize
between machines.

================================================================================
Sublime Text
================================================================================
1. Install Package Control plugin:
    https://sublime.wbond.net/
2. Install Package Syncing Plugin:
    https://sublime.wbond.net/packages/Package%20Syncing
3. Define Sync directory as {$SYNC_REPO}/sublime
4. Victory!
